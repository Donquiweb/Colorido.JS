Colorido.Js is a awesome, flexible, fast and cross-browser dynamically color changing js plugin for web projects. Read Instructions

Colorido is based upon following css classes. Add these classes in your web document to observe it

*Update: Added Dynamic Color Gradients

.colorido
Dynamic Background Color

.colorido-o
Dynamic Background Opacity

.colorido-t
Dynamic TextColor

.colorido-gr
Radial Gradient

.colorido-gl
Linear Gradient

.colorido-gd
Diagonal Gradient

.colorido-gh
Horizontal Gradient
